# Coding Convention

## Naming Convention

- **File Name:** Camel case.
- **Variable:** Camel case.
- **Function, Method, Class:** Pascal case.
- **CSS class:** BEM.
- **Get branch:** Snake case.

## JsonResult

Always include `success` property and `data` property. ( Can have other prop if needed ) 

*Example:*  

```json
{
   "success": true,
   "data": [],
}
